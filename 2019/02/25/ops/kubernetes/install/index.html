Install 安装的一些坑[Ubuntu]

  1. 需要注意的是主机hostname不能相同
  2. 发现网络插件无限重启那么一般是 /etc/resolv.conf文件出现问题
    • vi /etc/resolv.conf 添加上 nameserver 8.8.8.8
    • vi /etc/hosts 添加上 127.0.0.1 ${yourHostName}
  3. GFW的问题
    • Docker以及apt使用代理
    • 下载二进制文件，把对应的镜像转到其他仓库中
